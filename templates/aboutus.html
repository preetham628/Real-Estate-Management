PERFECT STAYS

This real estate system Give the functionality for customers, allowing them to search for property by features or address. It provides functionality for the seller, authorize them to look into the system and add new properties and meet the various customers needs. For this each property is provided with an agent to facilitate the customers to get connected with the owners/sellers.

Whenever searching is done for a new property, the main focus is on the location. As location being a spatial entity we are using the advantages given by spatial databases for our application. The application provides the customers to select any particular property and get information appropriately .Spatial data is data about location and space/dimensions
